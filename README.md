# Soil Moisture Sensor (ESP32) + I2C LCD (Wokwi)

Reads soil moisture from ESP32 analog pin GPIO 35 and displays the status on a 16x2 I2C LCD.

## Wiring (ESP32)
- LCD SDA -> GPIO 21
- LCD SCL -> GPIO 22
- Sensor AO -> GPIO 35

## Simulation (Wokwi)
All simulation files are in /simulation:
